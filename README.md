🛒 Olist E-Commerce Analysis

## Overview
End-to-end data analysis project on Olist Brazilian 
E-Commerce dataset using BigQuery, SQL, and Power BI.

## Key Insights
- 📦 96,000+ orders analyzed across 3 years
- 💰 Total Revenue: 13.22M BRL
- 🚚 Late Delivery Rate: 8.11% — structural issue 
  across all years
- 📈 Consistent YoY growth trend

## Business Recommendations
1. Open new warehouses in high late-rate regions
2. Focus marketing budget on Top 3 categories
3. Launch Q4 campaigns to offset seasonal drop

| Tool | Usage |

| SQL | 5 analytical views|
